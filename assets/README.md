# Assets

Drop your real images here, then swap the placeholder `<div class="frame">…</div>`
blocks in the HTML for real `<img>` tags. Each placeholder in the HTML has a comment
showing exactly what to write, e.g.:

```html
<!-- Replace with: <img src="assets/pints-home.png" alt="Pints of Joy homepage screen"> -->
<div class="frame frame--wide"><span class="frame__cap">Pints of Joy homepage — Figma</span></div>
```

Suggested file names (match what the comments reference):

- `hero-illustration.png` — your Adobe Illustrator hero (Home)
- `pints-home.png`, `pints-wireframes.png` — Pints of Joy
- `fittrack-home.png`, `fittrack-screens.png` — FitTrack
- `teesa-identity.png`, `teesa-packaging.png` — Teesa
- `sethi-grid.png` — Sethi Fine Jewelry
- `garima.jpg` — your About portrait
- `resume.pdf` — downloadable resume

Tip: keep images web-sized (long edge ~1600–2000px, compressed) so pages load fast.
